Интернет-магазин

Интернет-магазин — не корпоративный сайт, который можно рекомендовать практически всем компаниям. Интернет-магазин в первую очередь полезен владельцам стандартизированных товаров и услуг. Вторая группа потенциальных владельцев Интернет-магазинов — компании, занимающиеся традиционной розничной коммерцией.

Используемые мной технологии

  • БЭМ. Методология БЭМ создана в Яндексе для разработки сайтов, которые надо делать быстро, а поддерживать долгие годы. Она позволяет создавать расширяемые и повторно используемые компоненты интерфейса.
  • Шрифт в REM вместо PX. При изменении размера шрифта элемента для разных экранов все элементы на странице масштабируются пропорционально, то есть не теряется соотношение между размером букв, отступами в блоке, расстоянием между блоками и т. д.
  • Иконки в SVG. Формат svg занимают гораздо меньше места, чем их близнецы, такие как png, jpg, созданные как объект растровой графики. При помощи CSS можно быстро изменить параметр графики на сайте, например, цвет фона или позицию логотипа на странице.
